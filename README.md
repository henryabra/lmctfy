# Let Me Claude That For You

A passive-aggressive way to help people discover Claude AI.

**Live site: [henryabra.github.io/lmctfy](https://henryabra.github.io/lmctfy/)**

## What is this?

Inspired by "Let Me Google That For You" â€” when someone asks you a question that Claude could easily answer, send them a LMCTFY link instead.

The link plays a sarcastic animation showing them exactly how to ask Claude themselves.

## Usage

1. Enter the question on the homepage
2. Copy the generated link
3. Send it to your friend
4. Watch them learn a valuable lesson

## Example

```
https://henryabra.github.io/lmctfy/?q=how%20do%20I%20center%20a%20div
```

## Run locally

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

- Vanilla HTML/CSS/JS
